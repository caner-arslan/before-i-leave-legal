# Before I Leave - Legal Documents Website

This repository contains the legal documents and landing page for the Before I Leave mobile application.

## Files

- `index.html` - Landing page with app description and links to legal documents
- `privacy-policy.html` - Privacy Policy document
- `terms-of-service.html` - Terms of Service document
- `logo.png` - App logo (place your logo file here)

## Setup for GitHub Pages

1. Push all files to the `main` branch of this repository
2. Go to repository Settings â†’ Pages
3. Select source: "Deploy from a branch"
4. Select branch: `main` and folder: `/ (root)`
5. Click Save

Your site will be available at: `https://caner-arslan.github.io/before-i-leave-legal/`

## URLs for App Store Submission

- Privacy Policy: `https://caner-arslan.github.io/before-i-leave-legal/privacy-policy.html`
- Terms of Service: `https://caner-arslan.github.io/before-i-leave-legal/terms-of-service.html`

## Adding Logo

1. Copy your logo file (PNG format, preferably with white background) to this directory
2. Name it `logo.png`
3. The logo will automatically appear on the landing page

If the logo file is not found, a checkmark icon will be displayed as a fallback.

## Customization

You can customize the colors in the CSS by changing the gradient values:
- Primary green: `#22c55e`
- Dark green: `#16a34a`

These match the app's color scheme.
